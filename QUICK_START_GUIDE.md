# Quick Start - Production Ready Platform

## For Users

### Influencer Registration
1. Go to login page
2. Click "Join as Influencer"
3. Choose "Continue with Instagram"
4. Authorize the app
5. Profile auto-fills with Instagram data
6. Start browsing campaigns!

### Brand Registration
1. Go to login page
2. Click "Continue as Brand"
3. Create campaign
4. Browse and connect with influencers

### Admin Access
1. Go to login page
2. Click "Admin Access"
3. Use admin credentials

## Influencer Workflow

1. **Dashboard** - View performance metrics
2. **Browse Campaigns** - Find opportunities
3. **Apply** - Submit applications
4. **Upload Content** - Share deliverables
5. **Track Earnings** - View wallet balance
6. **Get Verified** - Unlock premium campaigns
